开发指南

有视通讯服务平台为您提供完整的资料文档以及清晰的案例资源， 帮助开发人员把所需的通信功能集成到网页或是移动应用中。

  • Javascript Library

    Javascript Library可将WebRTC功能轻松集成到web应用程序和网站中。 DEMO源码 查看更多开发文档

  • Android SDK

    Android SDK可将文本、音频、视频聊天等通信特性集成在iPhone，iPad等Android设备中。 下载Android SDK DEMO源码 查看更多开发文档

  • IOS SDK

    IOS SDK可将文本、音频、视频聊天等通信特性集成
    在以IOS为操作系统的iPhone，iPad以及其他移动设备中。 下载IOS SDK DEMO源码 查看更多开发文档
